# Dotfiles

## Installation

```bash
$ git clone git://github.com:teitei-tk/MyDotfiles.git /path/to/MyDotfiles
$ cd /path/to/MyDotfiles
$ make init
```

### Dotfiles

```bash
$ make setup # setup dotfiles and vscode settings
$ make setup-dotfiles # dotfiles only
```

### VSCode

```bash
$ make setup-vscode
```
